# Scribe2

长篇小说写作引擎。核心命题:连续写几百章不失忆、不涨上下文成本。

- **文件即记忆**:一书一目录,全部记忆是 markdown;git 即版本/快照/回滚。
- **检索即注入**:元素记忆(角色/世界书/伏笔/时间线/摘要/记录)走混合检索
  (实体名精确命中 + 向量相似 + 时近加权),固定层小而恒定。
- **提示词一等公民**:全部内置提示词在 `prompts/`,可整目录按书覆盖。

设计规格:`docs/SPEC.md`。

```
scribe new <书名>          # 对话建书
scribe write <书> 5        # 写第 5 章(流式)
scribe write <书> 5..30    # 连写
scribe audit <书>          # 一致性审查
scribe status <书>         # 就绪度/成本
```
