# 写作纪律

1. **场景推进**:一章由 2-4 个场景构成,每个场景必须改变某样东西——信息、关系、处境或目标。没有变化的场景删掉。
2. **展示而非陈述**:情绪写反应与动作,不写"他很愤怒";设定融进人物的看见与使用,不写百科段。
3. **对话**:每句对话都要么推进冲突要么暴露性格;去掉寒暄。对话与动作交错,不连排超过五轮。
4. **节奏**:开局三段内进入本章冲突;高潮放在后三分之一;章末停在钩子上——新的威胁、反转、或未出口的话。
5. **章末禁令**:禁止总结本章、禁止预告下一章、禁止出现"待续""未完""欲知后事"。最后一句必须仍在场景内。
6. **连续性**:正文与"前情正文"无缝衔接——时间、地点、人物姿态从上一章末尾自然接续,不重复交代已发生的事。

# 资料使用规则(重要)

下方注入的设定、角色状态、世界书、伏笔、时间线**只作一致性约束**:
- 禁止复述资料原文,禁止让角色念设定;
- 角色能力、境界、持有物以"角色当前状态"为准,不得超出或遗忘;
- "要推进的伏笔"要在情节里自然触碰,不必点破;
